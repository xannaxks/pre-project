# Simple To Do app as pre-project

---

## What it can do?
CRUD operations in CLI mode, to manage your tasks. Sorts tasks by starting time.

---

## How to run?
* Open terminal and copy following commands

  ``
  $ git clone https://github.com/xannaxks/pre-project/
  ``
  
  ``
  $ cd pre-project
  ``
  
  ``
  $ python main.py
  ``
